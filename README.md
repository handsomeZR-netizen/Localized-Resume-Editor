## ✨ 核心亮点 

### 🔒 本地化与隐私安全 

**您的数据完全属于您，永不离线。**

- **零数据上传**：所有编辑操作、图片上传和 PDF 生成过程完全在您的**本地浏览器**（Client-side）中完成。
- **无后端依赖**：我们不收集、不存储、不传输您的任何个人隐私数据（如电话、邮箱、工作经历等）。
- **安全无忧**：生成的 PDF 文件直接由浏览器渲染引擎构建，杜绝了第三方服务器泄露数据的风险。

### 👁️ 全局所见即所得 

告别繁琐的表单填写，直接像在 Word 中一样编辑网页。

- **点击即改**：页面上几乎所有的文字（姓名、简介、时间、经历描述等）都支持 `ContentEditable`，点击即可直接修改。
- **实时交互**：所有的样式调整和内容变更都会实时反馈，您看到的网页效果就是最终 PDF 的效果。

### 🎨 强大的动态定制能力

拒绝千篇一律的模板，打造专属简历。

- **自由增删**：支持动态添加“获奖经历”、“工作项目”和“技能标签”。
- **自定义板块**：不够用？点击底部按钮添加全新的自定义板块（例如：资格证书、个人爱好、开源贡献等）。
- **智能头像上传**：支持直接从电脑上传头像，自动适配圆形裁剪，且通过内联样式优化确保导出 PDF 时不变形。

### 📄 专业级 PDF 导出

- **一键生成**：内置高性能 PDF 渲染引擎，精准还原网页排版。
- **细节优化**：针对 PDF 导出特别优化了 SVG 图标和图片清晰度，修复了常见的图标丢失和拉伸问题。
- **自动清理**：导出时自动隐藏编辑按钮和操作栏，确保简历页面整洁专业。

## 🚀 快速开始 

此项目为单文件应用，部署极其简单。

1. **下载文件**：获取本项目的 `resume.html` 文件。
2. **运行**：直接双击使用 Chrome、Edge 或 Safari 浏览器打开。
3. **开始制作**：
   - **编辑**：点击文字修改内容。
   - **上传**：点击头像区域选择本地照片。
   - **添加**：点击底部的虚线框按钮添加更多经历。
   - **删除**：鼠标悬停在某个条目上，点击右上角的红色垃圾桶图标即可删除。
4. **保存**：点击底部的 **“导出为 PDF”** 按钮，保存到本地。

## 🛠️ 技术栈

- **HTML5**: 语义化标签构建文档结构。
- **Tailwind CSS**: 实用优先的 CSS 框架，实现快速响应式布局。
- **Vanilla JavaScript**: 原生 JS 实现 DOM 操作、事件委托与交互逻辑（无框架依赖，轻量极速）。
- **html2pdf.js**: 强大的客户端 PDF 生成库。
